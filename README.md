# Priyanka Majethiya GitHub Website

This is a static one-page academic website prepared for GitHub Pages.

## Files
- `index.html` — main website page
- `style.css` — website styling
- `assets/Priyanka_CV.pdf` — CV PDF
- `assets/profile-placeholder.svg` — placeholder profile image
- `.nojekyll` — helps GitHub Pages serve files directly

## How to publish on GitHub Pages
1. Create a public GitHub repository.
2. Upload all files from this folder.
3. If the repository name is `USERNAME.github.io`, the site will appear at:
   `https://USERNAME.github.io/`
4. Otherwise, go to **Settings > Pages** and deploy from the `main` branch.

## Local preview
You can open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`
